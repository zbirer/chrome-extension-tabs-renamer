# Privacy Policy for Tab Renamer

Tab Renamer does not collect, sell, share, or transmit personal data.

The extension stores user-created tab renaming rules locally in Chrome using Chrome extension storage. These rules may include URL patterns and tab title templates entered by the user.

The extension reads tab URLs and tab titles only locally in the browser to determine whether a saved rename rule should apply. This information is not sent to any external server.

All extension code runs locally in Chrome. The extension does not use remote code, analytics, tracking, advertising, or third-party data collection services.

If you have questions about this privacy policy, contact the repository owner at:

https://github.com/zbirer/chrome-extension-tabs-renamer
